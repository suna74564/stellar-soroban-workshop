# Frontend

React + TypeScript + Vite dApp arayuzu.

## Komutlar

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Ortam Degiskenleri

```bash
VITE_API_URL=http://localhost:3001
VITE_SOROBAN_RPC_URL=https://soroban-testnet.stellar.org
```

## Paketler

- `@stellar/freighter-api`
- `@stellar/stellar-sdk`
- `checkin` local generated Soroban binding
- `react`
- `vite`
